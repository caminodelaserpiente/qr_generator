# QR generator  :iphone: 


## Install  :gear:

```console
1. Clone this repo
    $ git clone https://github.com/caminodelaserpiente/qr_generator

2. Change the working directory
    $ cd qr_generator

3. Install the requirements
    $ pip install -r requirements.txt

5. Run the script
    $ python3 main.py
```


## Use :video_game:

```consol
python3 main.py
```
